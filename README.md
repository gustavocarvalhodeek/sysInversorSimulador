# sysInversorSimulador
Simulador web didático do inversor WEG CFW100 para treinamento em automação industrial, com HMI interativa, catálogo de parâmetros, rampas, falhas, motor simulado, presets, biblioteca de componentes, i18n PT-BR/EN-US, acessibilidade, testes automatizados, performance otimizada e foco em estudo técnico seguro.
